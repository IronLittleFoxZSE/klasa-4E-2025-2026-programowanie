2. Dodanie tabel

CREATE TABLE Person (
    Id INT NOT NULL AUTO_INCREMENT,
    Name VARCHAR(100) NOT NULL,
    Surname VARCHAR(100) NOT NULL,
    Age INT NOT NULL,
    PRIMARY KEY (Id)
);

4. Tworzymy projekt reposytorium dla bazy danych

5. Otwieramy folder z projektem repozytorium bazy danych (PPM na projekcie i otwieramy File Explorer; opcja pod koniec menu)

6. Otwieramy CMD i przechodzimy do folderu z projektem repozytorium bazy danych

7. Wykonujemy polecenie w CMD (TYLKO RAZ!!!!). Polecenie to instaluje rozszeżenie do platformy .NET.
dotnet tool install --global dotnet-ef

8. Do projektu repozytorium bazy danych dodajemy biblioteki. PPM na projekcie i wybieramy "Manage NuGet Packages..."

Microsoft.EntityFrameworkCore Version="9.0.13"
Microsoft.EntityFrameworkCore.Relational Version="9.0.13"
Microsoft.EntityFrameworkCore.Tools Version="9.0.13"
